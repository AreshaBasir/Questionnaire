# Questionnaire Description

This folder contains the questionnaire used for data collection in the study titled:

**"Exploring the Role and Reach of AI Chat Tools Among Engineering Students"**

## Purpose
The questionnaire was designed to examine how engineering students use AI-based chat tools for academic and learning purposes, as well as their perceptions regarding usefulness, limitations, and impact on learning behavior.

## Structure
The questionnaire consists of multiple sections, including:
- Demographic information
- Frequency and purpose of AI tool usage
- Perceived academic benefits and challenges
- Attitudes toward responsible and ethical use of AI tools

## Format
- The questionnaire was originally created using Google Forms.
- A PDF version is provided here for transparency and reproducibility.
